Desarrollé una versión del clásico juego Conecta 4 utilizando Java, ejecutándose completamente desde la consola. El proyecto permite que dos jugadores se turnen para colocar sus fichas, valida cada movimiento y detecta automáticamente cuándo un jugador gana o si la partida termina en empate. Este proyecto me permitió fortalecer mi lógica de programación, el manejo de matrices y la implementación de reglas de juego de forma clara y organizada.
